BR-004 enforcement: The blocklist message is intentionally generic — it does not name the rule, the source, or the destination. This prevents probing.
Sign-in upsell: The rate-limit copy nudges anonymous → owner conversion. Owners can be subject to a higher limit (out of scope here).
