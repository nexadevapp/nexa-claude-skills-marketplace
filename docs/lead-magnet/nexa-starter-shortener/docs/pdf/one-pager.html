Five failure modes of AI-generated code

#1 Silent assumptions — the agent guesses instead of asking
#2 Spec amnesia — forgets the spec mid-implementation
#3 Phantom abstractions — invents patterns nobody asked for
#4 Verification theater — tests pass but don't assert real behavior
