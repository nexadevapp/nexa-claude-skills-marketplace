UC-003 — Redirect to destination

Server-side 302 — no user-facing screen.

Backend GET /:slug
Why no screen: The redirect is an HTTP 302 issued from the server within p95 < 100ms (NFR-002). Showing an interstitial page would violate NFR-008 (no third-party trackers) by tempting analytics injection, and would frustrate users. The only failure path with UI is the 404 / expired page below.
